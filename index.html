Software QA Q&A

  1. What is Agile testing? Requirements, Design, Implemetation, Verification, Maintenance, and test before the application is complete so a pivot can be made easier.
  2. What waterfall testing? Scope is firm and fixed not easy to pivot when necessary.
  3. What is Bug release? Handing over software with known bugs.   What is Bug leakage? When testers failt to detect a bug before users report it.
  4. What is regression testing? Testing to see if a bug fix impacts other functionality.
  5. What is Crud or Black Box testing? Create, Read, Upadate, and Delete.
  6. What is Whitebox testing? It is transparent code based testing when you have access to the source code(Unit testing, Integration testing, regression testing. )
